# React + Vite

A minimal setup to get React working in Vite with HMR and ESLint rules without verbose start pages etc.

# Steps you should do to use this as your own template:

- git clone https://github.com/dyestuff/personal-react-template name-of-your-project
- cd name-of-your-project
- npm install
- npm run dev

# For example:

- git clone https://github.com/dyestuff/personal-react-template my-beautiful-project
- cd my-beautiful-project
- npm install
- npm run dev
